# as1

```
本次作业的任务是填写一个旋转矩阵和一个透视投影矩阵。给定三维下三个
点 v0(2.0, 0.0, −2.0), v1(0.0, 2.0, −2.0), v2(−2.0, 0.0, −2.0), 你需要将这三个点的坐
标变换为屏幕坐标并在屏幕上绘制出对应的线框三角形
```

## Run

```shell
$ mkdir build
$ cd build
```

```shell
$ cmake ..
$ make
```

```c++
./Rasterizer // 循环运行程序 创建一个窗口显示且你可以
// 使用 A 键和 D 键旋转三角形。

./Rasterizer −r 20 //运行程序并将三角形旋转 20 度然后将
// 结果存在 output.png 中

./Rasterizer −r 20 image.png //运行程序并将三角形旋转 20 度
// 然后将结果存在 image.png 中
```
